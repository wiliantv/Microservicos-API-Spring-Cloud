<h2>Serviços</h2>





- Basicas:
  1. **API Config** - Porta:`8888`
  2. **API Discovery** - Porta:`8761`

- Microsserviços
  - **Email** - Porta: 5000

- [**Spring Cloud Actuator**](https://spring.io/guides/gs/actuator-service/)
